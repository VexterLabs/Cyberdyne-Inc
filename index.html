Chat Completion

POST /api/chat

Generate conversational responses using state-of-the-art language models including Llama 3, Llama 2, and Mistral.

Parameters

Parameter Type Required Description
model string Yes Model to use (e.g., "llama-3-8b", "mistral-7b")
messages array Yes Array of message objects with role and content
temperature number No Controls randomness (0.0 to 1.0)
max_tokens integer No Maximum number of tokens to generate

Example Request

// JavaScript example using fetch
const response = await fetch('/api/chat', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'llama-3-8b',
    messages: [
      {role: 'user', content: 'Hello, how are you?'}
    ],
    temperature: 0.7,
    max_tokens: 150
  })
});

Text Embeddings

POST /api/embeddings

Generate vector embeddings for text input, useful for semantic search, clustering, and similarity comparisons.

Parameters

Parameter Type Required Description
model string Yes Embedding model to use
input string/array Yes Text or array of texts to embed

Example Request

const response = await fetch('/api/embeddings', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'text-embedding-ada',
    input: 'The quick brown fox jumps over the lazy dog.'
  })
});

Translation

POST /api/translate

Translate text between multiple languages with high accuracy and contextual understanding.

Parameters

Parameter Type Required Description
text string Yes Text to translate
source_lang string No Source language code (auto-detected if not specified)
target_lang string Yes Target language code (e.g., 'es', 'fr', 'de')

Example Request

const response = await fetch('/api/translate', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    text: 'Hello, how are you today?',
    target_lang: 'es'
  })
});

Text Summarization

POST /api/summarize

Generate concise summaries of long documents while preserving key information and context.

Parameters

Parameter Type Required Description
text string Yes Text to summarize
max_length integer No Maximum length of the summary in words
format string No Output format: 'paragraph', 'bullet_points'

Example Request

const response = await fetch('/api/summarize', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    text: 'Long document text goes here...',
    max_length: 150,
    format: 'bullet_points'
  })
});

Image Analysis

POST /api/image-analysis

Analyze images to extract information, recognize objects, scenes, and generate descriptive captions.

Parameters

Parameter Type Required Description
image string Yes Base64 encoded image or image URL
task string Yes Analysis task: 'caption', 'objects', 'tags'

Example Request

const response = await fetch('/api/image-analysis', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    image: 'data:image/jpeg;base64,/9j/4AAQSkZJRgABAQ...',
    task: 'caption'
  })
});
